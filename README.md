# CSS Live Preview PlayGround

⚠️ 현재 상태: 미완성(개발 진행 중)

## 프로젝트 소개: CSS Live Preview Playground

### 프로젝트 개요

이 프로젝트는 CSS 속성을 학습하며 실시간으로 확인하고 테스트하기 위한 프로젝트입니다. 핵심 CSS 주제들을 한 곳에 집중적으로 모아 제공하며 사용자가 값을 입력하는 즉시 대상 요소에 변화가 적용되어 시각적 이해도와 편리성을 확보하기 위해 구현했습니다.
